Warframe Items Shop - 自作情報サイトの公開

About this project

大好きなゲーム「Warframe」のアイテム情報を集約した個人ショップ風サイトを制作しました。 デザインの構築からコーディング、データの整理まで全て一貫して行っています。

「絶対的安定」を求めた自作PC構成とこだわり

System Concept

長時間の動画エンコードや複雑なMOD環境下でも決して揺るがない、**『絶対的な安定性』**を最優先したビルドです。 あえて定格運用を貫くことで、クリエイティブワークに集中できる環境を構築しました。

CPU

Intel Core i7-14700F

内蔵GPUなしの「F」を選択し描画負荷を分離

GPU

Radeon RX 6800 XT

VRAM容量に余裕のあるAMD上位モデル

Memory

64GB (Micron Tech)

2133 MT/s 定格運用の究極安定志向

Motherboard

ASRock B760 Pro RS/D4 WiFi

Total Storage

2.33 TB
